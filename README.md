# LG U+ 7기 docker 저장소 

이 리포지토리는 부트캠프 과정 중 **PySpark를 이용한 빅데이터 분석 실습 노트북 모음**입니다. Spark의 기초부터 머신러닝까지 다양한 주제를 Jupyter Notebook으로 학습하고 정리했습니다.

## 1. 학습 로드맵 & 노트북 구성

아래는 각 노트북의 순서와 주요 주제입니다.

| 파일명 | 주요 내용 |
|--------|-----------|
| `00_hello_pyspark.ipynb` | PySpark 환경 설정 및 첫 실행 ('Hello PySpark') |
| `01_start_spark.ipynb` | SparkSession 및 SparkContext 초기화 |
| `02_RDD_test.ipynb` | RDD 생성, 변환(Transformation), Action 개념 실습 |
| `03_KVDD.ipynb` | 키-값(pair) RDD 다루기 |
| `04_Operation.ipynb` | 다양한 RDD/DF 연산 (필터, 맵, 집계 등) |
| `05_SparkSQL_DF.ipynb` | DataFrame 생성 및 Spark SQL 연산 |
| `06_SparkDataAnal.ipynb` | DataFrame 기반 데이터 분석 및 요약 |
| `07_Classification.ipynb` | PySpark MLlib을 이용한 분류 모델 실습 |
| `08_Regression_1.ipynb`, `08_Regression_2.ipynb` | 회귀 모델 구현, 비교 및 성능 분석 |
---
## 2. 프로젝트 개요
이 프로젝트의 목표는 Spark의 핵심 개념과 데이터 처리 기법을 체계적으로 학습하는 것입니다.
RDD와 DataFrame, Spark SQL, 머신러닝까지 이어지는 실습을 통해 대규모 데이터 처리와 분산 분석 역량을 기반으로 한 실전 감각을 익힐 수 있습니다.
